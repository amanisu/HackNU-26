# 🎤 Что делать прямо сейчас

## Почему не работает "hello can you please draw a violet Circle"

**Ответ**: Нет Gemini API ключа! 🔑

Это **естественное сложное** команда на английском, которая требует AI для понимания.

## Решение (3 клика)

### 1️⃣ Получите API ключ (5 минут)

Откройте: https://aistudio.google.com/

- Кликните **"Get API key"**
- Выберите **"Create API key in new project"**
- Скопируйте ключ вида `AIzaSy...`

### 2️⃣ Введите ключ в приложение

1. Откройте https://localhost:5174
2. В onboarding форме найдите поле **"Gemini API Key"**
3. Вставьте скопированный ключ
4. Нажмите сохранить

### 3️⃣ Попробуйте снова

```
"Draw a violet circle"
"Create a red rectangle"
"Make a blue triangle"
```

## ✨ Что сейчас работает БЕЗ API ключа

Но для естественного языка типа **"draw a violet circle"** — нужен ключ.
