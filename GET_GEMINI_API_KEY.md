# 🔑 Как получить Gemini API ключ

## Шаг 1️⃣: Перейдите на Google AI Studio

Откройте: https://aistudio.google.com/

## Шаг 2️⃣: Создайте новый API ключ

1. Кликните на **"Get API key"** (справа вверху)
2. Выберите **"Create API key in new project"**
3. Подождите пока создается проект

## Шаг 3️⃣: Скопируйте ключ

1. Скопируйте показанный API ключ (нажмите copy icon)
2. Ключ должен выглядеть как: `AIzaSy...`

## Шаг 4️⃣: Введите в приложение

### Способ 1: В интерфейсе приложения (рекомендуется)

1. Откройте https://localhost:5174
2. Когда появится форма onboarding - введите ключ в поле **"Gemini API Key"**
3. Сохраните (кликните кнопку)

### Способ 2: В .env файле

1. Откройте файл `/Users/diana/hacknu/.env`
2. Добавьте строку:

```
VITE_GEMINI_API_KEY=AIzaSy...ВАШ_КЛЮЧ_ЗДЕСЬ
```

3. Перезагрузите страницу

## ✅ Как проверить что работает

Откройте Developer Tools (F12 → Console) и выполните:

```javascript
// Проверьте что ключ загружен
const { prefs } = usePrefsContext?.();
console.log(prefs?.geminiApiKey);
// Должно вывести: "AIzaSy..." (а не undefined)
```

## ⚠️ Важно

- **Бесплатно**: Google дает 15 000 запросов/день
- **Безопасно**: API ключ только для этого приложения
- **Никому не давайте**: Не делитесь ключом в интернете

## 🚫 Если ошибка "API key not set"

Значит ключ не загрузился. Проверьте:

1. ✅ Ключ введен в onboarding форму
2. ✅ Ключ в формате `AIzaSy...`
3. ✅ Нет пробелов в начале/конце
4. ✅ Перезагружена страница (Cmd+R)
5. ✅ Проверьте в console что `prefs.geminiApiKey` не пусто
